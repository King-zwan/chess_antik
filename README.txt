Παιχνίδι Σκάκι σε C++ (Object-Oriented)

Περιγραφή:
Αυτό το έργο είναι ένα πλήρως λειτουργικό παιχνίδι σκάκι μέσω κονσόλας, το οποίο αναπτύξαμε στο πλαίσιο του μαθήματος Αντικειμενοστραφούς Προγραμματισμού στο πανεπιστήμιο. Υλοποιεί όλους τους κανόνες του σκακιού και βασίζεται πλήρως σε αντικειμενοστραφή σχεδίαση.

Χαρακτηριστικά:

Πλήρης αρχικοποίηση σκακιέρας

Κανόνες κίνησης για όλα τα πιόνια (πιόνι, πύργος, ίππος, αξιωματικός, βασίλισσα, βασιλιάς)

Υποστήριξη ειδικών κανόνων:

1)Ροκέ (Castling)

2)En Passant

3)Προαγωγή Πιονιού

4)Έλεγχος για Σαχ και ΜΑΤ

5)Εισαγωγή κινήσεων μέσω κονσόλας (π.χ. e2e4)

6)ASCII εμφάνιση της σκακιέρας στο terminal

7)Υποστήριξη δύο παικτών (λευκά / μαύρα)

Τεχνικές λεπτομέρειες:

Χρήση κλάσεων για κάθε τύπο κομματιού

Κλάσεις Board και Player για οργάνωση λογικής

Δυναμική διαχείριση μνήμης για αιχμαλωσίες και προαγωγές

Προσομοίωση κινήσεων για έλεγχο ασφάλειας βασιλιά

Δώστε τα ονόματα των παικτών και εισάγετε κινήσεις όπως:
e2e4
g8f6
